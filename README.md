# Meeting Protocol Creator 📝🎤

Автоматизированная система создания профессиональных протоколов совещаний из аудиозаписей с использованием ИИ.

---

## 🚀 Что нового в Версии 2.0
- **Локальный ИИ (Offline Mode)**: Теперь можно работать без Yandex Cloud! Поддержка **Ollama** и **Faster-Whisper**.
- **Умные таблицы**: Поручения и решения теперь ГАРАНТИРОВАННО оформляются в виде наглядных Markdown-таблиц.
- **Оптимизация под CPU**: Настройка под легкие модели (Qwen 2.5 3B + Whisper Medium/Small) для работы на обычных ноутбуках.
- **GPU Ускорение**: Добавлен отдельный конфиг для владельцев видеокарт NVIDIA.
- **AI-Аудитор**: Встроенная проверка точности протокола вторым проходом ИИ.

## 🚀 Быстрый старт (Docker)

1.  **Настройка:** Отредактируйте ackend/.env. Выберите AI_PROVIDER=local для работы дома или yandex для облака.
2.  **Запуск (CPU):**
    `ash
    docker-compose up -d --build
    `
3.  **Запуск (GPU):**
    `ash
    docker-compose -f docker-compose.yml -f docker-compose.gpu.yml up -d --build
    `

## ✨ Основные возможности
- **Мировые стандарты:** Протоколы по правилам международного делового оборота.
- **Умные таблицы:** Поручения автоматически упаковываются в Word/Markdown таблицы.
- **Авто-диаризация:** Разделение спикеров по смыслу или голосу.
- **Email-рассылка:** Отправка готового файла на почту сразу после генерации.
