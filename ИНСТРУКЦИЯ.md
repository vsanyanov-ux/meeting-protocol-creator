# ИНСТРУКЦИЯ ПО ЗАПУСКУ (v2.0)

## 🛠 Быстрый запуск
1. Убедитесь, что Docker Desktop запущен.
2. Используйте Запустить_Протоколы.bat для старта всех сервисов.

## 🤖 Переключение ИИ (Облако / Офлайн)
Теперь вы можете менять «мозги» системы одним кликом:
- switch_to_yandex.bat — Переключает на Yandex Cloud (быстрее, выше качество).
- switch_to_local.bat — Переключает на локальный запуск (приватно, бесплатно).

## 📄 Где искать результаты
Все готовые протоколы (Word) сохраняются в папку: backend/temp_protocols/
