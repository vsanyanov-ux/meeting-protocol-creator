# Протоколист: ИНСТРУКЦИЯ ПО ЗАПУСКУ (v3.0 - Windows Native)

## 🛠 Быстрый запуск (Windows)
1. Если запускаете впервые: запустите **Установить_Зависимости.bat**.
2. Убедитесь, что приложение **Ollama** запущено (для локального ИИ).
3. Используйте **Запустить_Протоколы.bat** для старта системы.

## 🤖 Переключение ИИ (Облако / Офлайн)
Вы можете менять активного провайдера одним кликом:
- **switch_to_yandex.bat** — Переключает на Yandex Cloud (высокое качество).
- **switch_to_local.bat** — Переключает на локальный запуск (приватно, бесплатно).
- **switch_to_sber.bat** — Переключает на GigaChat от Сбера. (Добавить если нужно)

## 📄 Где искать результаты
Все готовые протоколы (Word) сохраняются в мамку: **backend/temp_protocols/**
Интерфейс доступен по адресу: http://localhost:5177

> [!NOTE]
> Docker больше не требуется для работы системы в этом режиме.
