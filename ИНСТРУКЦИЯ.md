# Протоколист: ИНСТРУКЦИЯ (v4.2.1)

## 🛠 Быстрый запуск
1. Если запускаете впервые: запустите **Установить_Зависимости.bat**.
2. Убедитесь, что приложение **Ollama** запущено.
3. Используйте **Запустить_Протоколы.bat** для старта.

## 🎙️ Новинка: Распознавание спикеров (Диаризация)
В версии 4.2.1 добавлена функция разделения голосов. 
- Для работы требуется `HF_TOKEN` в файле `backend/.env` (токен HuggingFace с доступом к pyannote/speaker-diarization-3.1).
- При загрузке файла в интерфейсе поставьте галочку **"Диаризация"**.
- Система сначала разделит голоса с помощью акустической модели, а затем ИИ попытается восстановить имена участников из контекста.

## 🤖 Переключение ИИ
- **switch_to_local.bat** — Локальный запуск (Whisper + Ollama).
- **switch_to_yandex.bat** — Yandex Cloud.
