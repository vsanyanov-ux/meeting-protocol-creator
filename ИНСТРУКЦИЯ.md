# ИНСТРУКЦИЯ ПО ЗАПУСКУ (v2.0)

Для запуска системы выполните следующие шаги:

1. Откройте терминал в этой папке.
2. Убедитесь, что Docker запущен.
3. Запустите файл Запустить_Протоколы.bat или введите:
   `ash
   docker-compose up -d
   `
4. Откройте в браузере: http://localhost:90

Приятного использования!
