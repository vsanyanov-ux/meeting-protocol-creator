# История изменений

Все значимые изменения в этом проекте будут фиксироваться в данном файле.

## [5.0.0] — 2026-04-27
### Добавлено
- **Обновленный интерфейс v5.0.0**: 
    - Новые индикаторы контуров безопасности (Закрытый/Открытый) с иконками статуса.
    - Опциональная отправка на Email: добавлен переключатель в интерфейсе.
    - Ускоренная синхронизация: частота обновления статусов увеличена для плавного отображения прогресса.
    - Информационные бейджи: в шапке теперь отображается активная модель и тип контура.
- **Whisper Turbo**: Полный переход на модель `large-v3-turbo`. Скорость транскрипции увеличена в 2.5 раза.
- **Qwen 3.5 (9B)**: Переход на новейшую архитектуру для качественного анализа встреч.
- **Управление VRAM**: Внедрена система «горячей замены» моделей (`keep_alive: 0`) для стабильности на 12 ГБ VRAM.
- **Точные Таймстемпы**: Улучшена навигация по времени в транскрипте.

### Изменено
- **Конфигурация по умолчанию**: Whisper Turbo и Qwen 3.5 установлены как стандартный стек.
- **Архитектура очистки**: Оптимизирован жизненный цикл объектов `faster-whisper` для предотвращения крашей на Windows.

### Исправлено
- **Langfuse Compatibility**: Исправлена ошибка `AttributeError` в Langfuse SDK v4.0.6 путем оптимизации механизма проброса атрибутов сессии.
- **Стабильность длинных файлов**: Исправлен сбой при обработке аудио более 20 минут.

## [4.4.0] — 2026-04-27
### Добавлено
- **Langfuse сессии**: Добавлено персистентное отслеживание сессий (`session_id`) в Langfuse для лучшего мониторинга.

### Изменено
- **Скрытие технической информации**: Технический вывод JSON скрыт из пользовательского интерфейса.
- **Очистка кода**: Удалены временные скрипты и кэш базы данных статусов (SQLite).

## [4.3.0] — 2026-04-24
### Добавлено
- **Универсальные промпты**: Полный переход на нейтральную роль «Профессионального секретаря», что исключило технический уклон и галлюцинации.

### Изменено
- **Формат протокола**: Раздел «НЕРЕШЕННЫЕ ВОПРОСЫ» теперь всегда содержит текст и является финальной точкой документа.
- **Чистота вывода**: Отчет AI-Аудитора теперь используется только для внутренней аналитики.
- **Запрет иноязычных вставок**: Усилены инструкции по предотвращению появления иероглифов в ответах.

## [4.2.1] — 2024-04-24
### Исправлено
- **UI Progress Sync**: Устранена проблема с «откатом» прогресс-бара до 0%.
- **SQLite Persistence**: Окончательный переход на SQLite для хранения статусов.
- **Live Transcription Progress**: Добавлено отображение количества обработанных фрагментов речи.

## [4.2.0] — 2024-04-24
### Добавлено
- **Multi-worker Backend**: Бэкенд переведен на работу через 2 воркера (Uvicorn).
- **Atomic Persistence**: Реализована система атомарного сохранения статуса и чанков.
- **Cross-process GPU Locking**: Файловая блокировка `gpu.lock` для координации ресурсов GPU.
