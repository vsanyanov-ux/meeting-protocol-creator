# 📓 Журнал Мастерства: Эволюция «Протоколиста»

Этот журнал фиксирует ключевые технические победы, ошибки и архитектурные сдвиги. Здесь хранится опыт, купленный временем и отладкой.

---

## 🏛️ Эпизод I: От Диаризации к «Бессмертному Бэкенду» (23 Апреля 2026)

### 🔴 Проблема (The Wall)
При интеграции локальной диаризации (Pyannote) и транскрипции (Whisper) сервер на Windows + Python 3.13 начал «падать» с **Exit Code 1** сразу после завершения обработки. 
- **Симптом**: Код отрабатывал, результат был, но процесс Python схлопывался в момент очистки памяти.
- **Причина**: Несовместимость деструкторов CUDA-библиотек (`ctranslate2`, `torch`) с циклом завершения потоков Windows в Python 3.13.

### 🧪 Попытки и тупики
1. **Async-to-Thread**: Попытка запустить в отдельном потоке внутри FastAPI не помогла — падал весь процесс.
2. **Sequential Cleanup**: Ручная очистка кэша (`torch.cuda.empty_cache`) лишь оттягивала падение, но не решала коренной конфликт.

### 💡 Прорыв (The World Model Shift)
Мы перешли от концепции «одного большого процесса» к **Изоляции Процессов (Process Isolation)**.
- **Решение**: Создан `transcribe_worker.py` — отдельный скрипт-камикадзе.
- **Архитектура**: Основной сервер (`main.py`) запускает воркера через `subprocess`. Даже если воркер упадет в самом конце (при очистке GPU), бэкенд выживает.

### 🎓 Уроки Мастерства (Lessons Learned)
1. **Изоляция важнее интеграции**: Если библиотека ведет себя нестабильно на уровне ОС (DLL, CUDA), не пытайся «подружить» её с сервером — вынеси её в отдельный процесс. Это основа паттерна «Микросервисы в одном флаконе».
2. **GPU Indexing**: На машинах с двумя GPU (AMD + NVIDIA) библиотеки часто выбирают «индекс 0» (встроенную графику). Всегда явно задавай `CUDA_VISIBLE_DEVICES=1` в `.env`. Это экономит часы «необъяснимых» тормозов.
3. **VRAM Orchestration**: 12 ГБ VRAM — это много, но не бесконечно. Держать Whisper Medium и Qwen 2.5 одновременно можно только при чистом окружении.
4. **UX/UI Прозрачность**: Пользователь должен видеть, что система делает. Отображение имени файла и реальных этапов (а не заглушек про «облако») повышает доверие к продукту.

- **Model**: Qwen 2.5 (7B) + Whisper Medium.
